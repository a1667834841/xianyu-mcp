# 闲鱼爆款商品分析

## Overview

用于分析闲鱼同品类竞品，提取高曝光商品的价格、标题关键词、描述结构和成交信号，并产出可直接发布的标题与描述。

核心原则：先分析竞品，再写文案；不要把账号登录排查当成分析前置步骤。

## When To Use

- 用户要分析闲鱼爆款、竞品、同行商品、价格区间、关键词策略。
- 用户要参考热卖商品改写标题、描述、卖点或发布文案。
- 用户要找高曝光商品的共同规律，如低价词、清仓理由、信任背书、痛点表达。

## Tool Boundaries

| 场景 | 使用 | 不要默认使用 |
| --- | --- | --- |
| 搜索竞品 | `xianyu_search` | `xianyu_browser_overview` |
| 获取商品详情 | `xianyu_get_detail` | 登录二维码、浏览器状态检查 |
| 工具返回会话/登录错误 | `xianyu_check_session`，必要时 `xianyu_browser_overview` | 在没有错误时先查登录 |
| 复制发布商品 | 改用 `xianyu-skill` | 在本 skill 内扩展发布流程 |

红旗：如果用户只是要“分析爆款/优化文案”，不要先说“需要登录”“没有二维码工具”或调用浏览器状态工具。先搜索；只有工具明确报会话异常再排查登录。

## Workflow

1. 确定品类关键词；必要时扩展 2-4 个同义词。
2. 用 `xianyu_search` 搜索 30-100 条结果。
3. 按 `exposure_score`、`want_cnt`、价格和标题相关性选 TOP 5-10。
4. 用 `xianyu_get_detail` 获取代表商品详情。
5. 提取价格带、标题词、痛点、信任背书、清仓理由、售后限制。
6. 输出分析结论和优化后的标题、描述。

## Analysis Checklist

| 维度 | 看什么 |
| --- | --- |
| 价格 | 主流价格带、包邮、低价锚点 |
| 标题 | 商品名、品牌/型号、规格、核心卖点、低价词 |
| 描述 | 来源说明、成色、痛点解决、信任背书、售后说明 |
| 图片 | 实拍、细节、包装、防伪、使用场景 |
| 成交信号 | `exposure_score`、`want_cnt`、评论/收藏线索 |

常见高转化词：全新、正品、包邮、现货、清仓、闲置、官网、防伪、支持验货、母婴店、实体店、最后几件。

## Copywriting Rules

- 标题控制在 20-30 个字左右，优先放完整商品名、规格和核心卖点。
- 描述必须分段；每段之间空一行，不要挤成一整段。
- 每段只表达一个卖点：来源、成色/规格、痛点、信任、价格、售后。
- 不使用 emoji、颜文字、闲鱼表情标签或 Markdown 加粗/列表。
- `【】` 只放在标题或首句，正文不要反复使用。
- 不编造品牌、授权、防伪、原价、功效、库存或发货承诺。

## Output Format

```markdown
竞品结论：
- 价格带：...
- 高频关键词：...
- 高曝光共性：...

优化标题：
...

优化描述：
第一段：来源或成色说明。

第二段：核心卖点或痛点解决。

第三段：信任背书或规格细节。

第四段：价格、发货、售后说明。
```

## Example

```text
优化标题：
【全新正品】泡泡玛特星星人蜂蜜毛绒挂件

优化描述：
全新未拆封，泡泡玛特官方正品，盒卡齐全，适合自留或送礼。

蜂蜜造型辨识度高，毛绒手感软糯，挂包、钥匙、手机都合适。

支持防伪码验真，实拍图可补，介意细节可以先确认再拍。

现货包邮，48小时内发出。盲盒类商品拆封不退，介意慎拍。
```

## Common Mistakes

| 误区 | 修正 |
| --- | --- |
| 默认先检查登录或浏览器状态 | 先执行搜索；只有工具报会话错误再检查 |
| 文案挤成一整段 | 按卖点分段，段落间空一行 |
| 复制竞品原文 | 提取结构和关键词，重新表达 |
| 只堆关键词 | 保留自然语序，优先说明真实卖点 |
| 编造背书 | 只写商品详情和用户提供的信息能支持的内容 |
