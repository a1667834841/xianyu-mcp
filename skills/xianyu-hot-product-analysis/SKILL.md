# 闲鱼爆款商品分析工作流

## Overview

**目的**：分析闲鱼热门商品，识别用户痛点、关键词策略和曝光驱动因素。

**核心方法**：搜索 → 排序 → 详情 → 分析 → 总结

---

## 工作流程

```
Step 1: 搜索商品
    ↓
Step 2: 按曝光度排序取 TOP 10
    ↓
Step 3: 获取商品完整详情
    ↓
Step 4: 分析单个商品（痛点+关键词+优势）
    ↓
Step 5: 总结共性规律
```

---

## Step 1: 搜索商品

**工具**：`xianyu_search`

**参数**：
- `keyword`: 品类关键词（如"母婴"、"美妆"、"数码"）
- `user_id`: user-001 或 user-002
- `rows`: 100

**关键词扩展策略**：
```
母婴 → 母婴、婴儿用品、宝宝用品、童装、纸尿裤
美妆 → 美妆、护肤品、面膜、防晒霜
数码 → 数码、手机配件、耳机
```

**执行命令**：
```bash
./scripts/mcp-dev call xianyu_search --user-id user-001 --keyword "母婴" --rows 100
```

---

## Step 2: 按曝光度排序取 TOP 10

**操作**：在搜索结果中提取 `items` 数组，按 `exposure_score` 降序排序。

**关键指标**：
- `exposure_score`: 曝光度评分（越高越火）
- `want_cnt`: 想要数
- `price`: 价格
- `item_id`: 商品 ID

**输出表格**：

| 排名 | 商品标题 | 曝光度 | 价格 | 想要数 | 发布时间 |
|------|----------|--------|------|--------|----------|
| 1 | 23元包邮食品级爬爬垫处理 | 2949.75 | ¥23 | 12659 | 2025-02-08 |
| 2 | 熊猫布布吸奶器配件套装全新 | 2197.22 | ¥28 | 3667 | 2025-10-28 |
| ... | ... | ... | ... | ... | ... |
| 10 | 白菜价英版花王拉拉裤 | 438.91 | ¥20.5 | 1860 | 2025-02-13 |

---

## Step 3: 获取商品完整详情

**工具**：`xianyu_get_detail`

**参数**：
- `user_id`: user-001
- `item_url`: `https://www.goofish.com/item?id=<item_id>`

**批量获取**：逐一调用 TOP 10 商品的详情。

**获取字段**：
- `title`: 完整标题
- `description`: 商品描述
- `category`: 分类
- `brand`: 品牌
- `image_urls`: 图片列表
- `is_free_ship`: 是否包邮

---

## Step 4: 分析单个商品

**分析维度表**：

| 维度 | 分析内容 | 方法 |
|------|----------|------|
| **价格** | 价格区间？比市场价低多少？ | 对比同类商品 |
| **痛点解决** | 解决了什么用户担忧？ | 从描述提取"无味道"、"不红屁屁"等 |
| **信任背书** | 如何证明可信？ | 识别"母婴店货源"、"正品"、"品牌" |
| **清仓理由** | 低价理由是什么？ | 提取"房租到期"、"清仓处理" |
| **关键词** | 标题包含哪些高转化词？ | 拆解标题词汇 |
| **图片特点** | 主图展示方式？ | 实拍/堆叠/场景 |

**痛点识别关键词**：
```
安全痛点：无味道、可啃可舔、食品级
功能痛点：不红屁屁、不起坨、吸水量大
价格痛点：亏本卖、白菜价、半价出售
信任痛点：母婴店货源、正品保证、品牌
```

**爆款文案公式**：
```
[低价词] + [清仓理由] + [痛点解决] + [信任背书]
例：23元包邮 + 母婴店处理库存 + 无味道可啃可舔 + 能进商场品质
```

---

## Step 5: 总结共性规律

**统计维度**：

| 特征 | 计算方式 |
|------|----------|
| 价格区间 | MIN-MAX 价格范围 |
| 包邮率 | 包邮商品占比 |
| 清仓理由率 | 描述含"清仓"的占比 |
| 品牌背书率 | 品牌/母婴店货源占比 |

**爆款公式权重**：
```
高曝光 = 性价比(35%) + 痛点解决(25%) + 信任背书(20%) + 清仓理由(15%) + 文案技巧(5%)
```

**输出格式**：
```markdown
## 今日分析总结

**品类**：母婴
**TOP 10 价格区间**：¥17-46
**包邮率**：100%
**清仓理由率**：80%

**共性痛点解决**：
1. 安全担忧 → "无味道"、"食品级材质"
2. 功能担忧 → "不红屁屁"、"不起坨"
3. 价格担忧 → "母婴店清仓"、"房租到期"

**高曝光关键词**：
- 低价词：包邮、白菜价、亏本卖
- 信任词：母婴店货源、正品保证
- 清仓词：清仓处理、房租到期
```
